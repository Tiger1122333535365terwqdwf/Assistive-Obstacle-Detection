# 智能障碍物检测系统 - 用户指南

## 🎯 系统概述

这是一套有初中学生开发专为视障人士、老人、儿童设计的**实时障碍物检测系统**。
麻烦您填写此问卷以帮助我们改进项目：https://forms.cloud.microsoft/Pages/ResponsePage.aspx?id=dlPdK4RTuUOkbgVnOkdsRUnPCo4F1cRBp9lUfpjwLu9UN0JLUDNJUjlNMlVWRDIxM0dUMlpaUVVKNi4u
通过 AI 视觉识别技术，系统能够：
- ✅ 实时检测周围障碍物
- ✅ 自动语音警告提醒
- ✅ 支持多种用户身份配置
- ✅ 提供友好的图形界面

## 🚀 快速开始

### 第一步: 启动应用

**最简单的方法：** 双击 `启动菜单.bat`

```
请选择操作:
[1] 启动 GUI 应用 (桌面版) ← 推荐选择这个
[2] 启动 Web 应用 (网页版)
[3] 运行系统测试 (验证)
[4] 查看文档
[5] 打开文件夹
```

### 第二步: 配置系统

**左侧面板中选择:**

1. **用户身份** (根据使用者选择)
   - 老人模式：语速慢，信息详细
   - 儿童模式：语言简单，友好提醒
   - 视障人士：位置信息，完整描述
   - 标准模式：标准语速和表达

2. **计算设备**
   - CPU：兼容所有电脑
   - GPU：需要 NVIDIA 显卡（更快）

3. **推理后端** (保持默认即可)
   - ONNX FP16（推荐）：快速且准确

4. **其他选项**
   - 如无摄像头：勾选"仅语音模式"

### 第三步: 启动检测

点击绿色的 **[启动] 启动检测** 按钮

**系统会：**
1. 加载 AI 模型
2. 打开摄像头
3. 开始实时检测
4. 显示视频画面（可选）
5. 播放语音警告

### 第四步: 停止检测

完成后点击红色的 **[停止] 停止检测** 按钮

## ❓ 常见问题

### Q: 启动后看不到摄像头画面
**A:** 
1. 摄像头可能未连接或被占用
2. 检查其他应用是否在使用摄像头
3. 尝试启用"仅语音模式"运行
4. 重启计算机后重试

### Q: 没有听到语音提醒
**A:**
1. 检查系统音量（Windows 任务栏右下角）
2. 确保没有静音
3. 检查是否选中"仅语音模式"
4. 试听系统声音（Windows 设置 → 声音）

### Q: 检测不准确
**A:**
1. 确保光线充足
2. 摄像头镜头保持清洁
3. 尝试移动到不同位置
4. 某些物体可能难以识别（这是正常的）

### Q: 应用很慢
**A:**
1. 关闭其他应用节省资源
2. 尝试"ONNX FP16"后端（更快）
3. 检查 CPU/GPU 使用率
4. 降低视频分辨率

## 🔧 系统需求

- **操作系统**: Windows 10 或更新版本
